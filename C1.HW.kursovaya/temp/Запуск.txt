	Сборка и запуск
Для сборки выполните:

bash
make

Для запуска:

bash
make run

или

bash
./temp_app

Для очистки:

bash
make clean

	Особенности реализации:
temp_api.h - содержит все прототипы функций

temp_api.c - содержит реализации всех функций

main.c - основная программа с демонстрацией работы

Makefile - для автоматизации сборки

Функции статистики рассчитывают:

Среднемесячную/среднегодовую температуру

Минимальную температуру за период

Максимальную температуру за период

Программа выводит как исходные данные, так и статистику по месяцам и за год.
